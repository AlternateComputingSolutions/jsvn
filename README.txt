The goal of JSVN is to develop a Java Swing client to access and
manipulate a Subversion repository.  The JSVN project also
includes a set of Ant tasks and a plugin for IntelliJ's Idea.

The project homepage is http://jsvn.alternatecomputing.com/
The subversion homepage is http://subversion.tigris.org/
